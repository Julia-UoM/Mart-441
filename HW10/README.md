For this project, I created a webpage that uses jQuery AJAX to load data from a local JSON file and display it as a set of cards. I chose a yoga pose dataset because it was easy for me to understand and organize visually (and I need to memorize this data myself for my YTT, so two birds with one stone). I created a JSON file with pose information like name, type, benefit, and difficulty, then used jQuery to load that data and loop through it, adding each pose to the page.

One part of this assignment that was somewhat confusing to me was creating the custom jQuery plugin. I defined a plugin using $.fn that applies styling and a hover effect to each card. The plugin adds padding, a border, and a soft shadow, and it also changes the background color when the user hovers over a card. What I found interesting is that instead of styling each element individually, I could apply the plugin once to a group of elements and have the same effect applied to all of them. That helped me understand how plugins make code more reusable and efficient. Super interesting stuff!

I used the class examples and ChatGPT to help me understand how to structure the AJAX call and how to correctly loop through the JSON data. Once I understood the general pattern, it became easier to adapt it to my own dataset and layout. ChatGPT helped me to structure the final product. I also focused on making the layout feel visually intentional by using a soft pastel color palette inspired by last weeks living poster project, so the final result felt cohesive, although somewhat simple.
